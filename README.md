# 🌌 AURA Studio | The Infinite Creative Suite

A professional-grade vector drawing application redefined for the digital artisan.

## 🎨 Minimalist Power
AURA Studio combines cinematic aesthetics with a high-performance vector engine. Experience an unrestricted canvas designed for visionaries.

### ✨ Key Capabilities
- **Aura Transform Engine**: Professional 8-point resizing for shapes and text nodes.
- **Ethereal Text Suite**: High-fidelity inline editing with real-time typography scaling.
- **Cinematic Startup**: A high-impact brand experience driven by Framer Motion.
- **Pro HUDs**: Floating glassmorphism panels for a distraction-free worksheet.
